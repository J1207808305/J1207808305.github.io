Sakura

Sakura 是一个基于Cortex M4的小型物联网定位系统，致力于打造一个室内高精度定位兼物联网通信的系统平台，用于物联网通信节点，室内移动设备定位终端等场景。

硬件  开发环境  平台

  • WINDOWS10    Ubuntu 16.04
  • TM4C123GXL   DWM1000   MPU9250
  • ZEPHYR

开始你的开发

硬件准备

  • 开发板: TM4C123GXL:Tiva C 系列 TM4C123G LaunchPad 评估套件是用于德州仪器 (TI) 基于 ARM® Cortex™-M4F 的微控制器的低成本评估平台。你可以从官网或者淘宝等购买此开发板。
  • 定位模块: Thunb v1.0

从github获取源代码

Sakura 项目基于zephyr嵌入式系统开发,相关的移植工作还在进行中，下面提供的代码为定位的基本例程，相对应的开发环境为IAR。

{% highlight bash %} $ git clone https://github.com/SakuraProject/Sakura.git {% endhighlight %}

准备开发环境

安装IAR开发环境，进入Thunb/project目录下，双击thunb.eww开始项目的开发之旅。
